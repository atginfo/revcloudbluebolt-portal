# revcloudbluebolt-portal
Repository for Bluebolt Revenue Cloud Portal Initiative

## About
This package is meant to supplement Revenue Cloud processes with Partner Portal features for Account, Asset and Documents as well as displaying Invoice Line Items and Billing via Billing Schedules. Integration support via triggers and Netsuite are also included in this Salesforce metadata but will not be part of the unmanaged package. To view specific metadata by features, please view the [metadata-by-feature](revcloud-bluebolt/metadata-by-feature) folder.

## CMT Framework Accelerator
[Installation Link](https://login.salesforce.com/packaging/installPackage.apexp?p0=04tKj000000ihr1) 

1. Ensure the Salesforce Connected Apps, Docusign Apps Launcher and Salesforce CPQ packages are installed  

2. Click the link above and log into your specific Salesforce Org 

3. Ensure “Do not install.” under the options for “What if existing component names conflict with ones in this package?” 

4. Click “Install for All Users” 

5. Wait for Package to install, and then navigate to your Salesforce Org 

6. Navigate to “Installed Packages” under Setup, and verify the package install
